インストール
===================================

ROS2-TMS for Constructionインストールには、以下の2通りの方法があります。

1. スクリプトでインストール
   以下のコマンドを実行してください。::

         cd
         mkdir -p ~/ros2-tms-for-construction_ws/src && cd ~/ros2-tms-for-construction_ws/src
         git clone https://github.com/irvs/ros2_tms_for_construction.git
         cd ros2_tms_for_construction
         . setup.sh
      
2. 個々のコマンドをターミナル入力してインストール
   
   `こちら <https://github.com/irvs/ros2_tms_for_construction/blob/main/setup.md>`_  の手順にしたがってコマンドを実行してください
